Word By Word (2023)

Word By Word is an interactive two-player game built using C programming. The game is designed to test players' memory and recall abilities by presenting words one by one to form a complete sentence. Players must memorize and reproduce the sentence correctly to progress to the next level. The game increases in difficulty as the levels advance, encouraging players to improve their focus and cognitive skills.

ATM Simulation (2023)

ATM Simulation is a practical program developed in C++ that replicates the core functionalities of an automated teller machine (ATM). Text Compressor and Decompressor (2023)
Text Compressor and Decompressor (2023)

Text Compressor and Decompressor is a project that utilizes Huffman coding to compress and decompress text files. Huffman coding is an efficient algorithm that encodes characters based on their frequency of occurrence, significantly reducing file sizes without losing any data. This feature is particularly useful for optimizing storage and improving data transmission efficiency.

Developed in C++, the program consists of two main functionalities: a compressor that converts the input text into a compact binary format, and a decompressor that accurately reconstructs the original text from the compressed data. The program employs advanced data structures such as priority queues and binary trees, showcasing the practical application of algorithmic concepts taught in the course.
